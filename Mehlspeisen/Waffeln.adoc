= Waffeln

für 4 Personen

|===
|125g |Butter (weich) .4+| in Rührschüssel verrühren
|100g|Zucker
|1|Ei
|1Pk.|Vanillezucker
|250ml|Milch .3+|Milch komplett und Rest portionsweise zugeben, weiterrühren

|===
